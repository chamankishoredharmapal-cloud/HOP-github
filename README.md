# House of Padmavati

A digital sanctuary for heritage and contemporary luxury sarees, woven with quiet artistry.

## Tech Stack

- Vite 5
- TypeScript
- React 18
- shadcn-ui (Radix primitives)
- Tailwind CSS 3
- Supabase (PostgreSQL + Edge Functions)
- React Router v6
- TanStack React Query
- Razorpay (payments)

## Getting Started

```sh
npm install
npm run dev
```

The dev server runs at `http://localhost:8080`.

## Project Structure

```
src/
  assets/          — Images and static assets
  components/      — UI components (shadcn + custom)
  contexts/        — React contexts (Cart, Wishlist)
  hooks/           — Custom hooks
  integrations/    — Supabase client + types
  lib/             — Utility helpers
  pages/           — Route pages
  types/           — TypeScript type definitions
```
