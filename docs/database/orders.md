# Database: Orders & Order Items

## `orders`

| Column | Type | Default | Description |
|--------|------|---------|-------------|
| `id` | `uuid` PK | `gen_random_uuid()` | |
| `customer_id` | `uuid` FK → `customers(id)` | | |
| `shipping_address_id` | `uuid` FK → `shipping_addresses(id)` | | |
| `order_number` | `text` UK | `HOP-YYYYMMDD-XXXXXX` | Generated by `generate_order_number()` |
| `status` | `order_status` | `pending_payment` | Enum: `pending_payment`, `confirmed`, `processing`, `shipped`, `delivered`, `cancelled`, `returned` |
| `payment_status` | `payment_status` | `pending` | Enum: `pending`, `paid`, `failed`, `refunded`, `partially_refunded` |
| `subtotal` | `int` | | In paise (₹1 = 100 paise) |
| `shipping_cost` | `int` | `0` | |
| `total` | `int` | | subtotal + shipping_cost |
| `notes` | `text` | | Internal notes |
| `courier_name` | `text` | | Shipping carrier |
| `tracking_number` | `text` | | Shipping tracking |
| `cancellation_reason` | `text` | | Why cancelled |
| `cancelled_at` | `timestamptz` | | When cancelled |
| `created_at` | `timestamptz` | `now()` | |
| `updated_at` | `timestamptz` | `now()` | Auto-updated by trigger |

**RLS**: Enabled — zero policies. Returns empty for `authenticated` without explicit `SELECT` policies.

| Column | Type | Description |
|--------|------|-------------|
| `id` | `uuid` PK | |
| `order_id` | `uuid` FK → `orders(id)` ON DELETE CASCADE | |
| `product_id` | `text` | SKU or product reference |
| `product_name` | `text` | Snapshot at order time |
| `product_price` | `int` | Snapshot at order time (paise) |
| `quantity` | `int` | |
| `image_url` | `text` | Snapshot at order time |
| `created_at` | `timestamptz` | |

**RLS**: Same as orders — default-deny.

## Indexes

- `idx_orders_order_number` unique on `order_number`
- `idx_orders_customer_id` on `customer_id`
- `idx_orders_status` on `status`
- `idx_orders_payment_status` on `payment_status`
- `idx_order_items_order_id` on `order_id`

## Functions

### `generate_order_number()`
Returns `text` in format `HOP-YYYYMMDD-XXXXXX` using a PostgreSQL sequence.

### `create_order(p_customer_email, p_customer_name, ...)`
Transactional RPC that:
1. Find-or-create customer by email
2. INSERT shipping_address
3. INSERT order
4. INSERT order_items (from JSON array)
5. RETURN order_number + order_id

### `update_updated_at_column()`
Trigger function sets `NEW.updated_at = NOW()`. Applied to `orders` table.
