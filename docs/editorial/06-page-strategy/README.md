# Page Strategy

## Overview

Every page on House of Padmavati serves a specific purpose in the customer's journey. A page strategy document defines that purpose — the business goal, the user goal, the emotional intent, the reading depth, and the calls to action. Without a page strategy, content drifts. With one, every sentence earns its place.

This directory will contain one document per page, each following the standard template below.

When writing for any page, first read its strategy document. Understand the primary emotion, the System 1 hook, and the System 2 anchor before choosing a single word.

## Page Strategy Template

Each page strategy document follows this structure:

```
# Page Strategy: {Page Name}

## Purpose
{Why this page exists in one sentence}

## Business Goal
{What this page must accomplish for HOP}

## User Goal
{What the woman visiting this page wants}

## Primary Emotion
{The dominant feeling this page should create — from the emotional journey}

## Secondary Emotion
{The supporting feeling that reinforces the primary}

## Story Goal
{What story does this page tell the woman about herself?}

## Trust Goal
{What trust does this page need to build or reinforce?}

## SEO Goal
{What search intent does this page serve?}

## Reading Depth
{How deeply will the woman read? Scanning? Skimming? Absorbing?}

## Reading Time
{Estimated time to read all content on the page}

## Content Priority
{What the woman should see first, second, third — in order of importance}

## Primary CTA
{The single most important action on this page}

## Secondary CTA
{Supporting action — optional, never competing}

## System 1 Hook
{The sensory or emotional element that triggers desire}

## System 2 Anchor
{The rational justification that removes friction}

## Relationship to Other Pages
{What page did she come from? Where will she go next?}

## Content Sources
{What content types feed this page — collections, products, journal, etc.}
```

## How to Use Page Strategy Documents

1. **Before writing:** Read the strategy document for the target page. Understand the primary emotion, the CTA, and the content priority.

2. **During writing:** Refer to the System 1 hook and System 2 anchor for the page. Ensure both are present in the content.

3. **After writing:** Verify that every element on the page serves the purpose defined in the strategy. If an element does not serve the purpose, remove it.

4. **During review:** The reviewer checks the content against the page strategy — does it create the intended emotion? Does it serve the business and user goals?
